Welcome to Concentration!

A memory game based on a classic card game using an ordinary deck of cards!

Rules

  • Click a card and then click another card to flip them over and see if they are a match. If they don't match try again!
  • You are matching ranks, not suits(King, Ace, 2, etc.)
  • Cards will disappear when you get a match.
  • Your goal is to find all of the pairs(26) before your time is up.
  • You don't have to wait for the cards to flip back over to try again, every second counts!

Note: Concentration is intended to be played on desktop (52 is a lot of cards! 😂)

About

Concentration was built in one week for the Mintbean hiring hackathon in August 2021 using JavaScript and Phaser, an HTML game framework. The landing pages were built using HMTL/CSS and were styled using Bootstrap and it is deployed on Netlify. The source code can be found on Github, here.
