Welcome to Concentration!

A memory game based on a classic card game using an ordinary deck of cards!

Rules

  • Click a card and then click another card to see if they are a match. If they don't match try again!
  • You are matching ranks, not suits(King, Ace, 2, etc.)
  • Your goal is to find all of the pairs(26) before your time is up.

Note: Concentration is intended to be played on desktop (52 is a lot of cards! 😂)

About

Concentration was built for the Mintbean hiring hackathon in August 2021 using JavaScript and Phaser, an HTML game framework. It is deployed on Netlify. the code can be found on Github, here.
